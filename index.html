Galileo
Web App | January 2017

Info

Built at Daemon Dash 2017.

Many University of Maryland students are familiar with VENUS, a website that outputs possible class schedules given certain parameters. We saw various issues with VENUS and wanted to give it an upgrade, so my team and I made Galileo.

PhoneBuzz
Web App | November 2017

Info

As a part of a technical challenge for LendUp, I was tasked with creating a web app that lets users "play" the game FizzBuzz through a phone call.

I built the app using Flask, the Twilio API, and SQLite. Users can either call my Twilio number or request a call to their phone (either instantly or after a delay). Once the call is initiated, the user enters a number through their keypad and hears the "Fizz Buzz" sequence from 1 to that number. All calls/numbers are stored in an SQLite database and the history is shown in a table. Users can replay any call if they wish.

Fireberry Pi
Web App | April 2017

Info

Built at Bitcamp 2016.

Every year, about 500 deaths occur every year due to detectable (and thus preventable) gas leaks. Fireberry Pi combines a web app, Android app, and hardware setup (Raspberry Pi / Arduino) to autonomously detect gas leaks in the user's home and send immediate notifications to the user's family members.

I helped work on the hardware scripts (Python) and the web/Android apps.
